# 1. Introduction
Explore Diving uses a serverless architecture.  
No accounts, no tracking, and no personal data collection.

# 2. Information Collection

## 2.1 Personal Data
The App does **not** request or collect:
- Name, phone number, or email  
- Device identifiers  
- Login information  
- Any personally identifiable information (PII)

---

## 2.2 Location Information
The App does **not** access or use your GPS location.  
All dive site coordinates come from the built-in offline database.

---

## 2.3 Local Storage
The App stores the following data locally:
- Favorites  
- Visited sites  
- Wishlist items  
- User preferences  

These data:
- Are stored locally in SQLite  
- Are never transmitted to external servers  
- Can be fully removed by uninstalling the App
